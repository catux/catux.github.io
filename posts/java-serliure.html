Java serà lliure

Escrit al 2006-11-13 00:00:00 per cpina

Finalment, el Java implementació de Sun està sent alliberat utilitzant la llicència GPL 2.

Són molt bones notícies per diverses raons. Un fet a destacar és que fan servir llicència GPL 2, i no una nova llicència de Sun feta "a mida" amb pecualitats estranyes (veure comentaris de la FSF sobre CDDL). Després, això permetrà incorporar el Java de Sun a les distribucions Linux, per defecte. També permetrà fer versions millorades de Java, utilitzar
lliurement els applets trobats a Internet, etc.

Sun ha fet força coses pel programari lliure. Els últims anys es pot destacar OpenOffice i OpenSolaris (d'aquella manera, la llicència no és compatible amb GPL, tot i ser lliure). Però també m'agradaria (opinió personal) fer una lectura crítica sense voler ser "aigua-festes" del moment que han fet aquesta alliberació: quan .NET i C# guanyen terreny a Java (amb el suport de Mono per entorns lliures), mig any aproximadament després que intentessin dir que alliberaven amb una llicència que no va ser acceptada per tothom (sortosament! sinó segurament ara no tindriem la llicència GPL), i també quan el projecte GCJ (compilador de Java de la FSF) estava ja arribant a uns nivells molt alts de compatibilitat. Sembla que alliberaven (i ben lliure!) o estaven perdent molt de mercat. Sigui com sigui: més val ara que mai!!

PD: Afegeixo dos enllaços, són comentaris d'en Ricardo Galli o comentaris que en Ricardo ha traduït de'n Richard Stallman. El primer apunt (molt bo la part final del seu escrit) i el segon apunt.

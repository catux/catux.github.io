Compatibilitat Windows - Linux

Escrit al 2004-04-10 00:00:00 per whats_up

Ja ha sortit la versió beta de "Windows Services for Unix 3.5" (actualitzada a 8 d'abril), que és una col·lecció d'eines pels administradors de xarxes Windows pensades per facilitar-ne la integració dins d'entorns heterogenis. Entre d'altres característiques permet la sincronització de contrasenyes entre entorns Unix (Solaris, HP-UX, AIX i Linux) i Windows (2000 Professional, 2000 Server, XP i Server 2003).
